# Entregável Java 01 


## Parte 1 - Histórico e Evolução do Java



### Introdução

Java é uma das linguagens de programação mais utilizadas no mundo. Desde sua criação, tornou-se fundamental para o desenvolvimento de aplicações web, sistemas corporativos, aplicativos móveis e diversos outros tipos de software. Sua principal característica é a portabilidade, permitindo que programas escritos em Java possam ser executados em diferentes sistemas operacionais sem necessidade de grandes modificações.


### Criação do Java

A linguagem Java foi criada em **1995** por uma equipe de desenvolvedores liderada por [James Gosling](https://pt.wikipedia.org/wiki/James_Gosling), juntamente com [Mike Sheridan](https://govinfo.library.unt.edu/acoas/nominations/sheridanbio.htm) e [Patrick Naughton](https://pt.wikipedia.org/wiki/Patrick_Naughton). O projeto foi desenvolvido dentro da empresa [Sun Microsystems](https://en.wikipedia.org/wiki/Sun_Microsystems).

Inicialmente, o projeto tinha o nome **Oak** e foi criado para o desenvolvimento de software em dispositivos eletrônicos e sistemas embarcados. Porém, com o crescimento da internet na década de 1990, a linguagem passou a ser utilizada principalmente para aplicações web.

Uma das ideias centrais do Java foi o conceito **“Write Once, Run Anywhere”** (Escreva uma vez, execute em qualquer lugar), que significa que o código Java pode ser executado em diferentes plataformas graças à Java Virtual Machine (JVM).


### Empresa responsável pelo Java

Até 2010, o Java era desenvolvido e mantido pela empresa [Sun Microsystems](https://en.wikipedia.org/wiki/Sun_Microsystems). No entanto, nesse mesmo ano, a empresa foi adquirida pela [Oracle Corporation](https://en.wikipedia.org/wiki/Oracle_Corporation), que passou a ser a responsável oficial pelo desenvolvimento e manutenção da linguagem.

Desde então, a Oracle continua lançando novas versões do Java, com melhorias de desempenho, segurança e novas funcionalidades para atender às necessidades da programação moderna.


### Objetivo do Java e impacto na programação

O principal objetivo do Java sempre foi criar uma linguagem simples, segura, portável e orientada a objetos, capaz de funcionar em diferentes dispositivos e sistemas operacionais.

O impacto do Java na programação moderna é enorme. Entre seus principais usos estão:

- Desenvolvimento de **aplicações corporativas**
- Sistemas **backend para web**
- Desenvolvimento de **aplicativos Android**
- Sistemas **bancários e empresariais**
- Aplicações em **computação em nuvem**

Além disso, Java ajudou a popularizar conceitos importantes da **Programação Orientada a Objetos** (POO), como **encapsulamento**, **herança** e **polimorfismo**.


### Principais versões do Java

Desde seu lançamento, várias versões do Java foram lançadas com melhorias importantes. Algumas das mais relevantes são:

- **Java 1.0** (1996)
  - Primeira versão oficial da linguagem. Introduziu o conceito da JVM e permitiu que programas rodassem em diferentes plataformas.

- **Java 5** (2004)
  - Uma das atualizações mais importantes. Introduziu recursos como Generics, Annotations, Autoboxing e o for-each loop, facilitando o desenvolvimento.

- **Java 8** (2014)
  - Considerada uma das versões mais revolucionárias do Java. Introduziu Expressões Lambda, Stream API e melhorias significativas na programação funcional.

- **Java 11** (2018)
  - Versão de suporte de longo prazo (LTS). Trouxe melhorias de desempenho, novos métodos para manipulação de strings e simplificações na execução de código.

- **Java 17** (2021)
  - Outra versão LTS muito importante. Introduziu melhorias de segurança, desempenho e novas funcionalidades na linguagem.

- **Java 21** (2023)
  - Versão recente com suporte de longo prazo. Trouxe avanços em concorrência, desempenho e simplificação da linguagem, tornando o Java mais moderno e eficiente.
