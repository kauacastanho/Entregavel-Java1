# Entregável Java 01 


## Parte 4 - Mercado de Trabalho para Desenvolvedores Java


### Salário médio no Brasil
Os salários de desenvolvedores Java variam conforme o nível de experiência (júnior, pleno e sênior). Com base em pesquisas recentes:

- **Júnior:** Entre R$ 3.000 e R$ 6.000 por mês
- **Pleno:** Cerca de R$ 6.000 a R$ 12.000 por mês, com média próxima de R$ 6.273
- **Sênior:** Entre R$ 12.000 e R$ 22.000 por mês, podendo chegar a valores maiores dependendo da empresa

---

### Áreas onde o Java é utilizado
O Java é uma linguagem extremamente versátil e utilizada em diversas áreas:

- Desenvolvimento Web (Back-end)
- Criação de APIs e sistemas web
- Aplicações Corporativas (Enterprise)
- Sistemas bancários, ERP e grandes sistemas empresariais
- Desenvolvimento Mobile (Android)
- Aplicativos para smartphones
- Sistemas Distribuídos e Microservices
- Arquiteturas modernas usadas por grandes empresas
- Big Data e Processamento de Dados
- Ferramentas de análise de grandes volumes de dados
- Tecnologias e frameworks mais utilizados

---

### Empresas costumam exigir conhecimento em diversas ferramentas do ecossistema Java:

- [Spring Boot](https://spring.io/projects/spring-boot)
  - Framework mais usado para desenvolvimento de APIs e aplicações web
- [Hibernate](https://hibernate.org/orm/)
  - Responsável pelo mapeamento objeto-relacional (banco de dados)
- [Maven](https://maven.apache.org/)
  - Gerenciamento de dependências e automação de build

---

O mercado para desenvolvedores Java no Brasil é bastante aquecido, com boas oportunidades e salários competitivos. Profissionais que dominam frameworks modernos e conceitos como APIs, microsserviços e cloud computing tendem a se destacar e alcançar melhores posições na carreira.
