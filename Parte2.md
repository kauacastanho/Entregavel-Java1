# Entregável Java 01 


## Parte 2 - Ambientes de Desenvolvimento (IDEs)

### [1. Eclipse](https://eclipseide.org/)

#### Descrição:
Uma das IDEs mais tradicionais para desenvolvimento Java, muito utilizada em projetos corporativos.

#### Vantagens:
- Gratuita e open source
- Grande quantidade de plugins
- Forte suporte para Java

#### Desvantagens:
- Interface considerada menos intuitiva
- Pode ser pesada dependendo da configuração

### [2. IntelliJ IDEA](https://www.jetbrains.com/idea/)

#### Descrição:
IDE moderna e muito popular, conhecida por sua inteligência de código e produtividade.

#### Vantagens:
- Autocompletar avançado
- Interface amigável
- Alta produtividade

#### Desvantagens:
- Versão completa é paga
- Pode consumir bastante memória

### [3. NetBeans](https://netbeans.apache.org/front/main/index.html)

#### Descrição:
IDE oficial mantida pela Apache, bastante utilizada por iniciantes.

#### Vantagens:
- Fácil de usar
- Já vem com ferramentas integradas
- Boa para iniciantes

#### Desvantagens:
- Menos recursos avançados que IntelliJ
- Comunidade menor

### IDE escolhida

A IDE que eu escolheria para estudar é o IntelliJ IDEA, pois ela oferece uma experiência mais moderna e produtiva. Seus recursos de autocompletar e análise de código ajudam muito no aprendizado, além de facilitar a identificação de erros rapidamente.
